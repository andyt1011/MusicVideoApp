# MusicVideoApp
- 在Android Studio中打开后可点击Run运行到设备上
## 两大功能由同一app完成，可以由用户选择进入哪个功能的页面
![aa0036c8908e5de3a63da88b8999714](https://github.com/andyt1011/MusicVideoSoftware/assets/106307018/133fbd05-bd6c-4782-ab8b-f9c8ff60d0e1)

## 音频
### 界面包含音乐列表、音乐播放器（可在同一界面实现）
![e75ca858adbbbe1a1655489851f68cc](https://github.com/andyt1011/MusicVideoSoftware/assets/106307018/969e776e-acdf-48be-b97e-3ef3e275796c)

### 音乐播放器支持播放/暂停、上一曲/下一曲（列表循环）
### 音乐播放器显示当前播放时间、歌曲总时间、歌曲时间进度条
### 音乐播放器支持时间进度条滑动调节播放进度
![0cddd43011b11abc15f9d8ccd4641e9](https://github.com/andyt1011/MusicVideoSoftware/assets/106307018/8b1b7cc6-34c7-479c-84c0-ff84df2a7b08)

## 视频
### 界面包含视频视频列表、视频播放界面
![efef574a11d59d4612318e7d3a50f71](https://github.com/andyt1011/MusicVideoSoftware/assets/106307018/53d22eec-82e1-4f93-9cc6-573d564acad1)

### 本地视频列表，使用Recycleview支持可滑动，需要显示每条视频的名称，点击条目可以跳转至视频播放界面进行播放，且无操作3s后自动隐藏
